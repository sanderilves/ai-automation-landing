---
description: Review the landing page before pushing
---

Review index.html and check for the following issues:

1. MOBILE: Any elements wider than the viewport, overflow issues, or content that breaks on small screens
2. CONSISTENCY: Font sizes, colors, spacing that don't match the rest of the page
3. LAYOUT: Any broken flex or grid layouts, misaligned elements
4. CONTENT: Typos, placeholder text, or missing content
5. LINKS: Any broken or missing href values on buttons and links

Report findings as a simple list grouped by category. If nothing is wrong say "All good — ready to push."
